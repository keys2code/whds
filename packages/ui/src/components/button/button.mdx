# Button

Triggers actions or navigates to new pages. Supports seven visual variants, eight sizes, and works seamlessly with leading or trailing icons.

## Import

```tsx
import { Button } from "@keys2design/whds-ui/components/button"
```

## Variants

Use `variant` to communicate the intent of the action.

| Variant | When to use |
|---|---|
| `default` | Primary action — one per view |
| `outline` | Secondary action alongside a primary |
| `secondary` | Alternative emphasis without border weight |
| `ghost` | Tertiary actions, toolbars, icon-only |
| `destructive` | Irreversible actions (delete, remove) |
| `destructive-outline` | Destructive but lower visual weight |
| `link` | Inline navigation that reads as text |

## Sizes

Eight sizes are available. The three named text sizes (`sm`, `default`, `lg`, `xl`, `xs`) and five icon-only variants (`icon`, `icon-xs`, `icon-sm`, `icon-lg`, `icon-xl`).

## With icons

Add icons inside the button's `children`. Use `data-icon="inline-start"` on leading icons and `data-icon="inline-end"` on trailing icons — this applies correct margin and sizing automatically.

```tsx
import { DownloadSimpleIcon } from "@phosphor-icons/react/dist/ssr"

<Button>
  <DownloadSimpleIcon aria-hidden="true" data-icon="inline-start" />
  Download
</Button>
```
